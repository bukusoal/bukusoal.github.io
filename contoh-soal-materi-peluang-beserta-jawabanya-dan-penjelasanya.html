2. buatlah contoh soal MTK berserta Jawabanya materi linear ​


Penjelasan dengan langkah-langkah:

Nilai maksimum f(x, y) = 5x + 4y yang memenuhi pertidaksamaan x + y ≤ 8, x + 2y ≤ 12, x ≥ 0, dan y ≥ 0 adalah ...

a. 24

b. 32

c. 36

d. 40

e. 60

PEMBAHASAN:

- x + y ≤ 8

ketika x = 0, maka y = 8 .... (0, 8)

ketika y = 0, maka x = 8 .... (8, 0)

- x + 2y ≤ 12

ketika x = 0, maka y = 6 .... (0, 6)

ketika y = 0, maka x = 12 .... (12, 0)

Sehingga, grafik dari pertidak samaan di atas adalah:

Kita cari dulu titik B, yaitu titik potong dua buah garis, yaitu:

subtitusikan y = 4 dalam x + y = 8

x + 4 = 8

x = 4 .... (4, 4)

Jadi, nilai fungsi obyektifnya adalah:

f(x, y) = 5x + 4y

- titik A (0, 6)

5x + 4y = 5.0 + 4.6 = 24

- titik B (4, 4)

5x + 4y = 5.4 + 4.4 = 20 + 16 = 36

- titik C (8, 0)

5x + 4y = 5.8 + 4.0 = 40

Jadi, nilai maksimumnya adalah 40.

JAWABAN: D

3. Buatkan 3 soal dan jawabanya materi tentang peluang


Jawaban:

1. Dua dadu dilambungkan bersama-sama. Peluang munculnya mata dadu yang pertama 3 dan mata dadu kedua lima adalah…

A. 6 / 36

B. 5 / 36

C. 4 / 36

D. 3 / 36

E. 1 / 36

Jawaban:

Merupakan peluang kejadian saling lepas:

P(3 dan 5) = P(3) x P(5) = 1/6 x 1/6 = 1 / 36

2.) Jika sebuah dadu dan sekeping mata uang dilempar undi satu kali bersama, maka peluang untuk memperoleh GAMBAR pada mata uang dan bilangan ganjil pada dadu adalah…

A. 1/12

B. 1/6

C. 1/4

D. 1/3

E. 1/2

Pembahasan

Merupakan peluang saling bebas, maka:

P(gambar dan ganjil) = P(gambar) x P(ganjil) = 1/2 x 3/6 = 3/12 = 1/4

Catatan

P(gambar) = nK / nS = 1/2

P(ganjil) = nK / nS = 3/6

3. Dua buah dadu dilempar undi bersama-sama. Peluang munculnya jumlah mata dadu 9 atau 10 ,yaitu …

A. 5 / 36

B. 7 / 36

C. 8 / 36

D. 9 / 36

E. 11 / 36

Pembahasan

Merupakan peluang kejadian saling lepas:

P (9 atau 10) = Peluang(9) + Peluang(10) = 4/36 + 3/36 = 7/36

Keterangan

nS (2 dadu) = 36

nK (9) = (3,6), (6,3), (4,5), (5,4) = 4

nK (10) = (4,6), (6,4), (5,5) = 3

Jadi:

P(9) = nK / nS = 4/36

P(10) = nK / nS = 3/36

komposisi fungsi

jika (f ± g)(x) = f(x) ± g(x)

f(x) = 3x² - 7x + 5 , g(x) = 2 - 3x dan h(x) = x² + 4x

a). menentukan nilai (f + g)(x)

(f + g)(x) = f(x) + g(x) = (3x² - 7x + 5) + (2 - 3x) = 3x² - 7x + 5 + 2 - 3x = 3x² - 7x - 3x + 5 + 2 = 3x² - 10x + 7

b). menentukan hasil dari (f - h)(3)

(f - h)(3) = f(x) - h(x) = (3x² - 7x + 5) - (x² + 4x) = 3x² - 7x + 5 - x² - 4x = 3x² - x² - 7x - 4x + 5 = 2x² - 11x + 5 = 2(3)² - 11(3) + 5 = 2(9) - 33 + 5 = 18 - 33 + 5 = –10

c). menentukan hasil dari 2f(x) - g(x) + 4h(x)

2f(x) - g(x) + 4h(x) = 2(3x² - 7x + 5) - (2 - 3x) + 4(x² + 4x) = 6x² - 14x + 10 - 2 + 3x + 4x² + 16x = 6x² + 4x² - 14x + 3x + 16x + 10 - 2 = 10x² + 5x + 8

31. jangan lupa beserta penjelasanya ya kak,soalnya aku masi blm paham sama materinya​


a. (f o g)(x) adalah fungsi komposisi dari f(x) dan g(x). Untuk menentukan (f o g)(x), kita harus mengganti x dalam f(x) dengan g(x). Sehingga:

(f o g)(x) = f(g(x)) = f(2x + 3) = (2x + 3)² - 3(2x + 3) + 1 = 4x² + 12x + 8

b. (f o g)(2) = f(g(2)) = f(2(2) + 3) = f(7) = 7² - 3(7) + 1 = 49 - 21 + 1 = 29

c. (g o f)(x) adalah fungsi komposisi dari g(x) dan f(x). Untuk menentukan (g o f)(x), kita harus mengganti x dalam g(x) dengan f(x). Sehingga:

(g o f)(x) = g(f(x)) = g(x² - 3x + 1) = 2(x² - 3x + 1) + 3 = 2x² - 6x + 5

d. (g o f)(-3) = g(f(-3)) = g((-3)² - 3(-3) + 1) = g(10) = 2(10) + 3 = 23

Jadi, (f o g)(x) = 4x² + 12x + 8, (f o g)(2) = 29, (g o f)(x) = 2x² - 6x + 5 dan (g o f)(-3) = 23
