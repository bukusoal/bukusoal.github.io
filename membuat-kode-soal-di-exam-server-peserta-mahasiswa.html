2. Buatlah 5 buah soal penentuan kode warna pada resistor beserta jawabannyabantu ya Kk :)​


Jawaban:

Contoh 1:

GELANG 1 = Cokelat

GELANG 2 = Hitam

GELANG 3 = Merah

GELANG 4 = Emas

NILAI :

= 1 Cokelat 0 hitam X 100 Merah toleransi 5%

=10 x100

= 1000Ω

1000Ω = 1KΩ 5%

Jadi nilai resistor dengan warna gelang coklat ,hitam merah dan emas adalah 1 kilo Ohm toleransi 5%

CONTOH 2 :

GELANG 1 = Kuning

GELANG 2 = Ungu/violet

GELANG 3 = Merah

GELANG 4 = Perak

NILAI :

= 4 kuning 7 Ungu X 100 Merah

= 47 x100

= 4700Ω

4700Ω = 4K7 toleransi perak 10%

Jadi nilai hambatan resistor diketahui sebesar 4K7.

Penjelasan:

Toleransi menunjukkan keakuratan sebuah resistor, artinya seberapa besar nilai aktualnya yang berbeda dari nilai teoritisnya. Semakin kecil nilai % maka semakin akurat/presisi dari nilai sebuah resistor tsb.

Setting Static IP

sudo update-rc.d -f NetworkManager remove

# sudo gedit /etc/network/interfaces

lalu masukkan parameter2 berikut :

auto eth0

address 192.168.1.100
netmask 255.255.255.0
network 192.168.1.0
broadcast 192.168.1.255
gateway 192.168.1.1

auto eth1
iface eth1 inet static
address 192.168.50.1
netmask 255.255.255.0
network 192.168.50.0
broadcast 192.168.50.255

# sudo gedit /etc/resolv.conf

15. jelaskan fungsi baris kode Dev tun dan push route-gateway 10.101.10.1 dalam konfigurasi server.conf server OpenVPN​


Fungsi baris kode Dev tun dan push route-gateway 10.101.10.1 dalam konfigurasi server.conf server OpenVPN​ adalah sebagai berikut:

Dev tun: Baris kode ini berfungsi dalam penggunaan TUN/TAP interface dalam PC kita, untuk koneksi dial-up layaknya modem GSM atau CDMA gunakan dev tun untuk melakukan IP route.push route-gateway 10.101.10.1: kode ini untuk menambahkan routing gateway dengan kode 10.101.10.1.Penjelasan

OpenVPN adalah aplikasi/perangkat lunak sumber terbuka. Tujuannya adalah untuk mengakses internet melalui virtual private network atau sering disingkat  VPN. OpenVPN sendiri bekerja dengan  membuat koneksi terowongan point-to-point (PTP) terenkripsi. Aplikasi ini menggunakan kunci pribadi dan sertifikat, atau lebih khusus lagi, nama pengguna dan kata sandi untuk otentikasi untuk membuat koneksi. Aplikasi ini tidak hanya memungkinkan Anda  untuk mengakses situs web yang diblokir, tetapi juga dapat menyembunyikan alamat IP Anda saat mengakses Internet.

20. Bagaimana cara membuat Master Server?


Edit file konfigurasi mysql
$sudo nano /etc/mysql/mysql.conf.d/mysqld.cnf atau untuk versi yang lama
$sudo nano /etc/mysql/my.cnf
Edit/tambahkan baris berikut di file konfirgurasi
bind-address = 192.168.1.1 atau * (address server master)
server-id = 1
log_bin = /var/log/mysql/mysql-bin.log
binlog_do_db = database1 (database yg mau direplikasi)

Kalau mau lebih dari satu database tinggal tambahkan baris lagi
binlog_do_db = database2
binlog_do_db = database3 dst
binlog_ignore_db = database4 (database yg tdk ikut direplikasi)
Refresh database mysql $sudo service mysql restart
Login ke Mysql
#mysql -u root -p
mysql>GRANT REPLICATION SLAVE ON *.* TO 'slave_user'@'%'
IDENTIFIED BY 'password'; (buat user replikasi)
mysql>FLUSH PRIVILEGES;

22. perhatikan baris kode berikut root@web-server 1 :~ #echo "web1" > /var / www / html / index. html fungsi dari baris kode tersebut adalah​


Jawaban:

echo "web1" > /var / www / html / index. html berfungsi untuk mengisi file index.html (index webpage) dengan tulisan web1.

Penjelasan:

Dalam Unix, command echo berfungsi untuk mengeluarkan input yang diterima (stdin) ke output yang dituju (stdout). Pada command diatas, inputnya adalah web1, sedangkan outputnya diarahkan ke /var/www/html/index.html melalui operator >. Maka, teks web1 akan diarahkan ke dalam file index.html

31. Apa yang di maksud dengan Web Server? beserta fungsinya..


Jawaban:

Mungkin maksud Anda adalah: Apa yang dimaksud dengan Web Server? beserta fungsinya..

Hasil Telusur

Cuplikan pilihan dari web

￼

Server atau Web server adalah sebuah software yang memberikan layanan berbasis data dan berfungsi menerima permintaan dari HTTP atau HTTPS pada klien yang dikenal dan biasanya kita kenal dengan nama web browser (Mozilla Firefox, Google Chrome) dan untuk mengirimkan kembali yang hasilnya dalam bentuk beberapa halaman

Jawaban:

Web server adalah perangkat lunak yang berfungsi sebagai penerima permintaan yang dikirimkan melalui browser kemudian memberikan tanggapan permintaan dalam bentuk halaman situs web atau lebih umumnya dalam dokumen HTML.

Fungsi Web Server adalah

Memastikan semua modul yang dibutuhkan  tersedia dan siap digunakan Membersihkan penyimpanan, cache, dan module yang tidak terpakai Melakukan pemeriksaan keamanan terhadap HTTP request yang dikirimkan browser

Penjelasan:

Namun, web server dapat mempunyai dua pengertian berbeda, yaitu sebagai bagian dari perangkat keras (hardware) maupun sebagai bagian dari perangkat lunak (software).

Jika merujuk pada hardware, web server digunakan untuk menyimpan semua data seperti HTML dokumen, gambar, file CSS stylesheets, dan file JavaScript. Sedangkan pada sisi software, fungsi web server adalah sebagai pusat kontrol untuk memproses permintaan yang diterima dari browser.
