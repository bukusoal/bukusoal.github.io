20. PKWU SEKOLAH MENENGAH ATASkelas 11Ngasal = Repot!​


Maka laba per produknya adalah Rp. 1.600,00.

Pembahasan:

Harga Pokok Produksi ( HPP )

Harga Pokok Produksi atau HPP adalah harga pokok dari suatu produksi. Jika dijual dengan harga tersebut, produsen tidak akan mengalami keuntungan maupun kerugian. Harga Pokok Produksi dapat dihitung menggunakan Rumus berikut.

=> Total Biaya / Jumlah Produksi

Penyelesaian:

Diketahui =

HPP = Rp. 8.000,00%Laba = 20%

Ditanya =

Besar laba per produk = ...?

Dijawab =

Besar laba per produk =

= HPP × %Laba= Rp. 8.000,00 × 20%= Rp. 8.000,00 × 20/100= Rp. 160.000,00/100= Rp. 1.600,00

Kesimpulan =

Jadi, laba per produknya adalah Rp. 1.600,00.

Jawaban:

Kreativitas dan keterampilan peserta didik dalam menghasilkan produk kerajinan, produk rekayasa, produk budidaya maupun produk pengolahan sudah dilatih melalu Mata Pelajaran Prakarya sejak di Sekolah Menengah Pertama Kelas VII, VIII dan Kelas IX. Peserta didik telah diperkenalkan pada keragaman teknik untuk menghasilkan produk kerajinan, produk rekayasaan, produk budidaya dan produk pengolahan. h(x) = f(g(x))

h(x) = 5 + 2(2x - 4)

h(x) = 5 + 4x - 8

h(x) = 4x - 3

2.

h^(-1)(4x - 3) = x

misalkan

(4x - 3) = a

4x = a + 3

x = (a + 3)/4

Jadi,

h^(-1)(a) = (a + 3)/4

Ubah a jadi x,

h^(-1)(x) = (x + 3)/4

3.

k(x) = 2(2x + 5) - 4

k(x) = 4x + 10 - 4

k(x) = 4x + 6

4.

k^(-1)(4x + 6) = x

misalkan.

(4x + 6) = a

4x = a - 6

x = (a - 6)/4

Jadi,

k^(-1)(a) = (a - 6)/4

Ubah a jadi x,

k^(-1)(x) = (x - 6)/4
