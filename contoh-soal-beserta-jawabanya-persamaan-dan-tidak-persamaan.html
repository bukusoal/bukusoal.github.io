17. contoh soal cerita exponen beserta jawabanya​


Tentu! Berikut ini adalah contoh soal cerita yang melibatkan eksponen beserta jawabannya:

Soal:

Seorang petani memiliki 10 biji benih jagung yang dapat tumbuh dan berkembang menjadi pohon jagung setiap tahunnya. Menurut pengamatan petani tersebut, setiap tahun pohon jagung tersebut dapat menghasilkan dua kali lebih banyak benih jagung dari tahun sebelumnya. Jika petani menanam benih jagung pada tahun ke-1, berapakah jumlah benih jagung yang dihasilkan pada tahun ke-5?

A. 80 benih jagung

B. 160 benih jagung

C. 320 benih jagung

D. 640 benih jagung

Jawaban:

Untuk menyelesaikan masalah ini, kita dapat menggunakan konsep eksponen. Jumlah benih jagung yang dihasilkan setiap tahunnya dapat dinyatakan sebagai pangkat dua karena setiap tahunnya jumlah benih jagung menjadi dua kali lipat.

Sebagai contoh, jumlah benih jagung pada tahun ke-1 adalah 10 biji, kemudian pada tahun ke-2 menjadi 2 * 10 = 20 biji, dan pada tahun ke-3 menjadi 2 * 20 = 40 biji, dan seterusnya.

Jadi, jumlah benih jagung yang dihasilkan pada tahun ke-5 adalah 2^4 (2 pangkat 4) dikali dengan 10 biji yang ditanam pada awalnya, atau 16 * 10 = 160 biji.

Oleh karena itu, jawaban yang benar adalah B. 160 benih jagung.

20. 5 Contoh Soal Persamaan Nilai Mutlak Linear Satu Variabel Dan Jawabanya


Contoh soal tentang persamaan nilai mutlak linear satu variabel, yaitu:

Himpunan penyelesaian dari |3x – 7| + 3 = 16 adalah ….Himpunan penyelesaian dari |x² – 2| = 7 adalah ….Himpunan penyelesaian dari |x + 1|² – 3|x + 1| + 2 = 0 adalah ….Himpunan penyelesaian dari |2x – 3| = |x + 3| adalah ….Himpunan penyelesaian dari |3x – 4| = |5 + x| adalah ….

Penjelasan dengan langkah-langkah

Definisi nilai mutlak.

[tex]|x| = \left \{\begin{array}{cc}x, \: jika \: x \geq 0 \\-x, \: jika \: x < 0 \end [/tex]

Persamaan nilai mutlak.

|f(x)| = a maka |f(x)|² = a²|f(x)| = |g(x)| maka |f(x)|² = |g(x)|²

Diketahui

|3x – 7| + 3 = 16 |x² – 2| = 7|x + 1|² – 3|x + 1| + 2 = 0|2x – 3| = |x + 3||3x – 4| = |5 + x|

Ditanyakan

Tentukan himpunan penyelesaiannya!

Jawab

Nomor 1

     |3x – 7| + 3 = 16

           |3x – 7| = 16 – 3

           |3x – 7| = 13

           (3x – 7)² = 13²

 9x² – 42x + 49 = 169

9x² – 42x – 120 = 0

 3x² – 14x – 40 = 0

(x + 2)(3x – 20) = 0

(x + 2) = 0 atau (3x – 20) = 0

        x = –2                    x = [tex]\frac{20}{3}[/tex]

Jadi himpunan penyelesaiannya adalah {[tex]\frac{20}{3}[/tex], –2}.

Nomor 2

                   |x² – 2| = 7

                  (x²  – 2)² = 7²

             x⁴ – 4x² + 4 = 49

     x⁴ – 4x² + 4 – 49 = 0

           x⁴ – 4x² – 45 = 0

        (x² – 9)(x² + 5) = 0

(x + 3)(x – 3)(x² + 5) = 0

(x + 3) = 0 atau (x – 3) = 0 atau (x² + 5) = 0

        x = –3               x = 3                  x² = –5

Untuk x² = –5 tidak ada nilai x yang memenuhi.

Jadi himpunan penyelesaiannya adalah {–3, 3}.

 

Nomor 3

|x + 1|² – 3|x + 1| + 2 = 0

Misal

p = |x + 1|

berarti persamaan di atas menjadi:

  p² – 3p + 2 = 0

(p – 2)(p – 1) = 0

(p – 2) = 0 atau (p – 1) = 0

        p = 2                  p = 1

Untuk p = 2

         |x + 1| = 2

        (x + 1)² = 2²

   x² + 2x + 1 = 4

   x² + 2x – 3 = 0

(x + 3)(x – 1) = 0

(x + 3) = 0 atau (x – 1) = 0

29. 10 soal persamaan kuadrat dan jawabanya


10 soal persamaan kuadrat dan jawabannya. Bentuk umum persamaan kuadrat adalah ax² + bx + c = 0, dengan a ≠ 0. Rumus diskriminan: D = b² – 4ac

Fungsi diskriminan dari persamaan kuadrat yaitu:

D > 0 memiliki 2 akar real yang berbeda D = 0 memiliki 2 akar real yang sama (akarnya kembar/sama) D < 0 tidak memiliki akar real (akarnya imajiner/khayal)

Jika x₁ dan x₂ adalah akar-akar persamaan kuadrat maka

x₁ + x₂ = [tex]-\frac{b}{a}[/tex] x₁ . x₂ = [tex]\frac{c}{a}[/tex]

Menentukan persamaan kuadrat

x² – (x₁ + x₂)x + x₁ . x₂ = 0

Pembahasan

10 soal persamaan kuadrat dan jawabannya

1) Himpunan penyelesaian dari persamaan kuadrat 4x² – 3x – 10 = 0 adalah …

Jawab

4x² – 3x – 10 = 0

4x² + 5x – 8x – 10 = 0

x(4x + 5) – 2(4x + 5) = 0

(4x + 5)(x – 2) = 0

(4x + 5) = 0 atau (x – 2) = 0

x = [tex]-\frac{5}{4}[/tex] atau x = 2

Jadi himpunan penyelesaiannya adalah {[tex]-\frac{5}{4}[/tex], 2}

2) Persamaan kuadrat x² + x – 2 = 0, akar-akarnya x₁ dan x₂ dengan x₁ < x₂. Nilai 2x₁ + 3x₂ = …

Jawab

x² + x – 2 = 0

(x + 2)(x – 1) = 0

(x + 2) = 0 atau (x – 1) = 0

x = –2 atau x = 1

karena x₁ < x₂ maka x₁ = –2 dan x₂ = 1

Jadi nilai dari 2x₁ + 3x₂ adalah

= 2x₁ + 3x₂  

= 2(–2) + 3(1)

= –4 + 3

= –1

3) Jika salah satu akar persamaan x² + (a + 1)x + (3a + 2) = 0 adalah 5, maka akar yang lain adalah …

Jawab

x² + (a + 1)x + (3a + 2) = 0

salah satu akarnya adalah 5, maka

5² + (a + 1)5 + (3a + 2) = 0

25 + 5a + 5 + 3a + 2 = 0

8a + 32 = 0

8a = –32  

a = –4

Jadi persamaan kuadrat tersebut adalah

x² + (a + 1)x + (3a + 2) = 0

x² + (–4 + 1)x + (3(–4) + 2) = 0

x² – 3x + (–12 + 2) = 0

x² – 3x – 10 = 0

(x – 5)(x + 2) = 0

(x – 5) = 0 atau (x + 2) = 0

x = 5 atau x = –2

Jadi akar lain dari persamaan kuadrat tersebut adalah –2

4) Hasil kali akar-akar persamaan kuadrat 6x² – 2x + 3 = 0 adalah ....

Jawab

x₁ . x₂  

= [tex]\frac{c}{a}[/tex]

= [tex]\frac{3}{6}[/tex]

= [tex]\frac{1}{2}[/tex]

5) Jika x₁ dan x₂ adalah akar-akar persamaan kuadrat 2x² + 3x – 6 = 0, maka nilai dari 2x₁x₂² + 2x₁² x₂ = …

Jawab

2x₁x₂² + 2x₁² x₂

= 2 x₁x₂ (x₂ + x₁)

= 2 . [tex]\frac{c}{a}[/tex] . ([tex]-\frac{b}{a}[/tex])

= 2 . [tex]\frac{-6}{2}[/tex] . ([tex]-\frac{3}{2}[/tex])

= 2 . –3 . ([tex]-\frac{3}{2}[/tex])

= 9

6) Jika x₁ dan x₂ merupakan akar-akar persamaan 3x² – 4x – 2 = 0, maka x₁² + x₂² = …

Jawab

x₁² + x₂²

= (x₁ + x₂)² – 2 x₁x₂

= ([tex]-\frac{b}{a}[/tex])² – 2 . [tex]\frac{c}{a}[/tex]

= ([tex]-\frac{-4}{3}[/tex])² – 2 . [tex]\frac{-2}{3}[/tex]

= [tex]\frac{16}{9} + \frac{4}{3} [/tex]

= [tex]\frac{16}{9} + \frac{12}{9} [/tex]

= [tex]\frac{28}{9} [/tex]

7) Akar-akar persamaan kuadrat 2x² + mx + 16 = 0 adalah α dan β. Jika α = 2β dan α, β positif maka nilai m = …

Jawab

α . β = [tex]\frac{c}{a}[/tex]

2β . β = [tex]\frac{16}{2}[/tex]

2β² = 8

β² = 4

β = 2

maka α = 2β = 2(2) = 4

α + β = [tex]-\frac{b}{a}[/tex]

4 + 2 = [tex]-\frac{m}{2}[/tex]

6 = [tex]-\frac{m}{2}[/tex]

12 = –m

m = –12  

8) Persamaan kuadrat yang akar-akarnya –2 dan ½ adalah …

x² – (x₁ + x₂)x + x₁ . x₂ = 0

x² – (–2 + ½)x + (–2) . ½ = 0

9) Akar-akar persamaan kuadrat x² – 3x + 7 = 0 adalah α dan β. Persamaan kuadrat baru yang akar-akarnya 2α dan 2β adalah …

Jawab

α + β = [tex]-\frac{b}{a} = -\frac{-3}{1}[/tex] = 3 α . β = [tex]\frac{c}{a} = \frac{7}{1}[/tex] = 7

2. Persamaan garis yang sejajar dengan garis 2x+3y+6 = 0 serta melalui titik (-2,5) adalah …

a. 2x+3y-4 = 0 c. 3y+2x-11 = 0

b. 2x-2y+16 = 0 d. 3y-2x-19 = 0

Pembahasan :

Persamaan garis yang sejajar dengan 2x+3y+6 = 0 artinya gradien garisnya sama. Maka kita tentukan dahulu gradiennya sebagai berikut.

2x+3y+6 = 0

⇔ 3y = -2x – 6

⇔ y = -2/3 x – 2

maka gradiennya = -2/3

sehingga persamaan garis tersebut secara umum adalah y = -2/3x+c

Karena garis tersebut melalui titik (-2,5), maka titik tersebut kita substitusikan pada persamaan untuk mendapat nilai c.

y = -2/3x + c

⇔ 5 = -2/3 (-2) + c

⇔ 5 = 4/3 + c

⇔ c = 5 – 4/3

⇔ c = 15/3 -4/3

⇔ c = 11/3

Jadi persamaan garisnya adalah

y = -2/3x + c

⇔ y = -2/3 x + 11/3

⇔ 3y = -2x + 11

⇔ 3y + 2x – 11 = 0 (jawaban c)

maka gradien (m) = -1/2

⇔ -m = -2

⇔ m = 2

atau secara mudahnya, jika tegak lurus maka gradien garisnya lawan dan kebalikannya. karena m dari garis x+2y+1 = 0 adalah -1/2 maka lawan dan kebaliannya yaitu 2.

jadi persamaan garis y = m (x+1) menjadi y = 2(x+1)

y = 2(x+1)

⇔ 4 = 2k + 2

⇔ 2k = 4-2

⇔ 2k = 2

⇔ k = 1 ( jawaban a)
