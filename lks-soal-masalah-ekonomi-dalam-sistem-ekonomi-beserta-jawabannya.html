39. jawaban soal lks kls 8 halaman 8


Jawaban :

OBSERVING AND ASSOCIATING

1). Watched

2). Stopped

3). Ended

4). Spent

5). Went

ASSOCIATING

1). (+) The students made an observation this morning.

(-) The students did not make an observation this morning.

(?) Did the students make an observation this morning?

2). (+) She knew about the trouble among the students.

(-) She did not know about the trouble among the students.

(?) Did she know about the trouble among the students?

3). (+) You forgot about the assignment.

(-) You did not forget about the assignment.

(?) Did you forget about the assignment?

4). (+) The merchants sold many kinds of goods in the night market.

(-) The merchants did not sell many kinds of goods in the night market.

(?) Did the merchants sell many kinds of goods in the night market?

5). (+) They walked around the field last night.

(-) They did not walk around the field last night.

(?) Did they walk around the field last night?

PEMBAHASAN

Perintah dari kedua pertanyaan tersebut adalah di ubah ke dalam bentuk simple past.

Simple past adalah tenses yang digunakan untuk menjelaskan suatu kegiatan yang terjadi di masa lampau.

Rumus :

(+) S + V2 + O

(-) S + did + not + V1 + O

(?) Did + S + V1 + O

Contoh :

Tiara bought some vegetables.Tiara did not buy some vegetables.Did Tiara buy some vegetables?

Untuk pertanyaan di bagian Observing & Association karena kalimat nya bentuk positif (+) semua. Maka kata kerja nya harus di ubah ke dalam bentuk verb-2.

Bagaimana cara mengetahui perubahan kata kerja tersebut? Cara nya adalah dengan melihat di daftar regular verbs (kata kerja beraturan) ataupun di irregular verbs (kata kerja tidak beraturan). Dan berikut ini adalah perubahan kata kerja di atas :

Watch => Watched => Watched

Stop => Stopped => Stopped

End => Ended => Ended

Spend => Spent => Spent

Go => Went => Gone

Make => Made => Made

Know => Knew => Known

Forget => Forgot => Forgotten

Sell => Sold => Sold

Walk => Walked => Walked
