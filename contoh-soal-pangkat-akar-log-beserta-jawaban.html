Perpangkatan adalah sebuah bentuk operasi dalam ilmu matematika yang boleh digunakan apabila sebuah bilangan dikali dengan bilangan yang sama lebih dari satu kali. Pangkat sendiri merupakan banyaknya bilangan yang dikali dengan bilangan yang sama. Letak pangkat sendiri berada diatas bilangan yang dikalikan tersebut.

Bilangan berpangkat merupakan perkalian berulang suatu bilangan, dimana bilangan dapat berpangkat bulat positif, nol, maupun bulat negatif.

Contoh :

A⁴ = a × a × a × aA³ = a × a × aA² = a × a

Contoh Perpangkatan 2:

2² = ( 2 x 2 ) = 43² = ( 3 x 3 ) = 94² = ( 4 x 4 ) = 165² = ( 5 x 5 ) = 25

Contoh Perpangkatan 3:

2³ = ( 2 x 2 x 2 ) = 83³ = ( 3 x 3 x 3 ) = 274³ = ( 4 x 4 x 4 ) = 645³ = ( 5 x 5 x 5 ) = 125

Contoh Perpangkatan 4:

2⁴ = ( 2 x 2 x 2 x 2 ) = 163⁴ = ( 3 x 3 x 3 x 3 ) = 914⁴ = ( 4 x 4 x 4 x 4 ) = 2565⁴ = ( 5 x 5 x 5 x 5 ) = 625

Contoh Perpangkatan 5:

2⁵ = ( 2 x 2 x 2 x 2 x 2 ) = 323⁵ = ( 3 x 3 x 3 x 3 x 3 ) = 2734⁵ = ( 4 x 4 x 4 x 4 x 4 ) = 1.0245⁵ = ( 5 x 5 x 5 x 5 x 5 ) = 3.125

» Bilangan berpangkat merupakan perkalian berulang suatu bilangan (asli) dengan bilangan itu sendiri beberapa kali sebanyak yg ditunjukkan oleh bilangan (asli) itu.

» Sifat Sifat bilangan berpangkat

[tex]\boxed{ {a}^{m} \times {a}^{n} = {a}^{m + n} } \\ \boxed{ {a}^{m} \div {a}^{n} = {a}^{m - n} } \\ \boxed{( {a}^{m} {)}^{n} = {a}^{m \times n} } \\ \boxed{(ab {)}^{n} = {a}^{n} \times {b}^{n} } \\ \boxed{( \frac{a}{b} {)}^{n} = \frac{ {a}^{n} }{ {b}^{n} } }[/tex]

» Jenis jenis bilangan berpangkat

Bilangan berpangkat Positif yaitu Pangkat nya positif, tidak ada min ( - )Bilangan berpangkat Negatif yaitu Pangkat nya ada min ( - )Bilangan berpangkat nol ( 0 ) yaitu pangkat nya tidak ada, atau bisa dibilang 1

» Bentuk akar Merupakan bilangan yg menunjukkan pangkat dari bilangan tersebut dibagi oleh indeks yang terdapat dalam akar.
