28. apakah perbedaan antara web application dan form application?​


Jawaban:

perbedaannya ada pada penggunaannya.

Penjelasan:

yaitu web application digunakan bergantung kepada interaksi pengguna, apakah itu kontribusi content (Youtube, Twitter, Facebook) atau dengan mengumpulkan data dari sumber lain untuk ditampilkan kepada pengguna (Mint, Google Analytics, Klout) atau gabungan dari keduanya.

sedangkan form application digunakan untuk membuat aplikasi di mana user interface utama adalah browser. tentu, ini termasuk aplikasi yang dimaksudkan akan tersedia secara publik melalui world wide web, seperti e-commerce aplikasi. web form memiliki kegunaan lebih dari sekedar membuat situs web, tetapi aplikasi lain yang meminjamkan diri ke (thin front end) juga, seperti buku pedoman karyawan berbasis intranet atau aplikasi manfaat.
