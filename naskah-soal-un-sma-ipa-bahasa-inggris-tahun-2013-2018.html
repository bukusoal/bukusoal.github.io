15. Penyelesaian dari pertidaksamaan 25log (x-3) + 25log (x+1) ≤ ½ adalah … (Ujian Nasional Matematika Tahun 2013) **Super Intensif UN (Ujian Nasional) SMA/MA IPA , Penerbit: Bisa!Publishing 2016, Penulis: Muhammad Amien & Team


25log (x - 3) + 25log (x + 1) ≤ 1/2

⇔ 25log (x - 3)(x + 1) ≤ 25log 25^1/2

⇔ 25log (x² - 2x - 3) ≤ 25log 5

Syarat:

(1) x - 3 > 0 → x > 3
(2) x + 1 > 0 → x > -1
(3) (x² - 2x - 3) ≤ 5

⇔ x² - 2x - 8 ≤ 0

⇔ (x - 4)(x + 2) ≤ 0

⇔ x = 4 atau x = -2

⇔ -2 ≤ x ≤ 4

dari (1), (2), dan (3) diperoleh 3 < x ≤ 4.

Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
= (6 - 12)/(2√4)
= -6/4
