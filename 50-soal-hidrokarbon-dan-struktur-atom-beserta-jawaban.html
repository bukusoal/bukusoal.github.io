Atom karbon (C) diklasifikasikan primer, sekunder, tersier, dan kuarterner bergantung dengan jumlah atom C terikat dengan atom C yang dimaksud. Atom primer adalah atom C yang terikat langsung dengan 1 atom C lainnya. Atom sekunder adalah atom C yang terikat langsung dengan 2 atom C lainnya. Atom C tersier adalah atom C yang terikat langsung dengan 3 atom C lain. Atom C kuarterner adalah atom C yang keempat tangannya terikat langsung dengan atom C lainnya.

Pembahasan
Atom C memiliki 4 tangan (4 elektron valensi) sehingga perlu mengikat 4 atom lain untuk memenuhi kaidah oktet (agar stabil).

Atom CH3 pada senyawa hidrokarbon mengikat 3 atom H dan 1 atom C lainnya sehingga atom CH3 merupakan atom C primer. Oleh karena itu, atom 1, 2, 6, 9, 10, 11, dan 13 merupakan atom C primer.

Atom CH2 pada senyawa hidrokarbon mengikat 2 atom H dan 2 atom C lainnya sehingga atom CH2 merupakan atom C sekunder. Oleh karena itu, atom 4 dan 8 merupakan atom C sekunder.

Atom CH pada senyawa hidrokarbon mengikat 1 atom H dan 3 atom C lainnya sehingga atom CH merupakan atom C tersier. Oleh karena itu, atom 3, 5, dan 12 merupakan atom C tersier.

Atom C pada senyawa hidrokarbon mengikat 4 atom C lainnya sehingga atom C merupakan atom C kuarterner. Oleh karena itu, atom 7 merupakan atom C kuarterner.

18. apakah perbedaan ikatan jenuh dan tidak jenuh pada senyawa hidrokarbon?jelaskan disertai contoh strukturnya


Kelas         : X
Pelajaran   : Kimia
Kategori     : Senyawa Hidrokarbon
Kata Kunci : hidrokarbon, jenuh, tak jenuh, alkana, alkena, alkuna, ikatan tunggal, ikatan rangkap dua, ikatan rangkap tiga

Pembahasan

Ciri-ciri senyawa hidrokarbon jenuh (alkana)
⇒ hanya mengandung ikatan tunggal antara atom-atom karbon, sehingga mengandung atom hidrogen dalam jumlah maksimum
⇒ rumus umum (deret homolog) alkana  [tex] C_{n} H_{2n+2} [/tex], dengan n adalah jumlah atom karbon
⇒ tidak reaktif
⇒ mengalami reaksi substitusi  
⇒ ditemukan dalam minyak bumi dan gas alam
Contoh alkana:
CH₃ - CH₂ - CH₃ ⇒propana
CH₃ - CH - CH₂ - CH₃ ⇒ 2-metil-butana
            |
         CH₃

Ciri-ciri senyawa hidrokarbon tak jenuh (alkena & alkuna)
⇒ alkena mengandung ikatan rangkap dua antara atom-atom karbonnya, sedangkan alkuna mengandung ikatan rangkap tiga antara atom-atom karbonnya; hal ini menyebabkan jumlah atom hidrogen yang lebih sedikit dibanding alkana
⇒ rumus umum (deret homolog) alkena adalah [tex] C_{n} H_{2n} [/tex], sedangkan alkuna adalah [tex] C_{n} H_{2n-2} [/tex] dengan n adalah jumlah atom karbon
⇒ sangat reaktif
⇒ mengalami reaksi adisi  
Contoh alkena dan alkuna:
CH₂ = CH - CH₃ ⇒ 1-propena
CH₃ - C ≡ C - CH - CH₃ ⇒ 4-metil-2-pentuna
                        |
                      CH₃  
Uji bromin cair dapat membedakan hidrokarbon jenuh atau tak jenuh.
⇒ Alkana tak bereaksi dengan bromin cair
⇒ Alkena dan alkuna bereaksi dengan bromin cair ditandai dengan munculnya warna merah

Penjelasan

• Alkana termasuk kedalam jenis Hidrokarbon jenuh, yang berarti sebuah senyawa organik dengan ikatan tunggal antara atom dari Hidrokarbon jenuh dengan hidrogen yang paling sederhana.

• Alkena merupakan sebuah senyawa hidrokarbon yang tidak jenuh dan dilengkapi dengan ikatan rangkap dua (–C=C–).

• Alkuna juga termasuk kedalam klasifikasi dari hidrokarbon tak jenuh. Namun didalam Alkuna ini memiliki satu atau lebih ikatan rangkap tiga antara atom karbon. Alkuna ini dilambangkan dengan rumus umum yaitu CnH(2n-2).

38. Sebutkan nama dari struktur Hidrokarbon tersebut! Jawaban disertai dengan penjelasan dilarang Copas.​


Nama senyawa dari struktur hidrokarbon pada soal:

5 - etil - 2,2,3,7 - tetrametil dekana.

Untuk rumus struktur lengkap dan penomorannya ada pada lampiran (dokumentasi penulis sendiri). Adapun bentuk 3D-view juga tampak pada lampiran kedua (putih = atom H, biru tosca = atom C).

Penjelasan dengan langkah-langkah

Penamaan dari struktur hidrokarbon pada soal diasumsikan mengambil sistem IUPAC atau sistem Stock. Senyawa hidrokarbon pada soal adalah senyawa alkana karena hanya ada ikatan kovalen tunggal saja antar atom karbonnya dengan asumsi bahwa "atom hidrogen atau H pada pertengahan rangkaian seharusnya dituliskan".

Berikut langkah-langkah penentuan nama senyawa tersebut:

Menentukan nama rantai induk.
Pada rumus struktur di soal, rantai induk dengan jumlah atom karbon (C) terpanjang adalah 10 rangkaian atom karbon (C). Maka nama rantai induk adalah dekana.
Catatan:
- C₃H₇ dapat digambarkan dengan:
- CH₂ - CH₂ -CH₃ .Memberi urutan nomor pada atom-atom karbon rantai induk dari yang paling dekat dengan cabang alkil, yaitu dari kiri ke kanan (mulai angka 1 sampai 10). Cek lampiran.Menentukan nama cabang.
Cabang/gugus metil (CH₃):
∴ Ada di atom C ke-2 (2 cabang).
∴ Ada di atom C ke-3 (1 cabang).
∴ Ada di atom C ke-7 (1 cabang).
Maka nama untuk cabang metil: 2, 2, 3, 7 - tetrametil.
Cabang/gugus etil (C₂H₅):
∴ Ada di atom C ke-5 (1 cabang).
Maka nama untuk cabang etil: 5-etil.Penamaan lengkap senyawa hidrokarbon pada soal.
Nama cabang + Nama rantai induk.
∴ 5 - etil - 2, 2, 3, 7 - tetrametil dekana.Catatan:
∴ Urutan nama cabang adalah sesuai alfabet (dalam bahasa Inggris). Misal methyl dan ethyl yang mana awalan "e" lebih dahulu dari "m" maka ditulis cabang etil dulu baru metil.
∴ Ingat ada tanda "-" untuk memisahkan angka dan huruf.
