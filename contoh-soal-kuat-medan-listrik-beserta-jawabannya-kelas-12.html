6. tuliskan rumus Medan listrik Dan potensial listrik Dan berikan contoh soap beserta jawabannya! tolong dibantu dong..


Rumus medan listrik :
E = F/Q
dengan
E = kuat medan listrik ( N/c )
F = Gaya ( N )
Q = muatan ( coulomb )
Contoh soal :
Dua muatan sejenis 4 x 10^-4 C dan 8x10^-4C terpisah sejauh 6cm( k = 9x10^9. Cari :
(a) Gaya Coulomb?
(b) Kuat medan listrik?

Jawab :
(a) F = k.Q1.Q2/r^2
= 9x10^9x4x10^-4x8x10^-4/(6x10^-2)^2
= 288 x 10 / 36 x 10^-4
= 8 x 10^5
(b) E = F/Q
E = 8x10^5/4x10^-4x8x10^-4
= 80 x 10^4/32x10^-8
= 2,5 x 10^12N/c
Rumus potensial listrik
V = W/Q
dengan
V = beda potensial ( Volt )
W = Usaha ( Joule )
Q = Muatan ( Coulomb )
Contoh soal :
Untuk memindahkan 25 C diperlukan 75 Joule Brp beda potensial?
V = W/Q
= 75/25
= 3 volt
Mohon maaf apabila ada kesalahan

19. sebuah muatan medan listrik jika diletakan 4cm dari pusat medan listrik memiliki kuat medan listrik sebesar 160N/C. berapakah kuat medan listrik muatan tersebut jika diletakkan 5cm dari pusat medan listrik?Jawab dengan caranya ya kak , makasih.

karena muatannya sama maka
E1/r1²=E2/r2²
160/(4x10-²)²=E2/(5x10-²)²

semoga faham.

LISTRIK STATIS

diketahui:
r1 = 4 cm
E1 = 160 N/C
r2 = 5 cm

ditanya:
E2?

jawab:
KUAT MEDAN LISTRIK BERBANDING TERBALIK DENGAN KUADRAT JARAK PUSAT MEDAN LISTRIK

Rumus
E2 / E1 = r1² / r2²
E2 = E1 . r1² / r2²
E2 = 160 . 4² / 5²
E2 = 160 . 16 / 25
E2 = 2560 / 25
E2 = 102,4 N/C

jadi kuat medan listriknya adalah 102,4 N/C

Medan Listrik
