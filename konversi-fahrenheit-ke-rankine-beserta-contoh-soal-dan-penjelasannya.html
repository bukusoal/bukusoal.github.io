Konversi Satuan Panjang

Satuan panjang biasa kita gunakan ketika mempertimbangkan panjang dari sesuatu. Entah itu benda, jalan, dan lain sebagainya.

Untuk melakukan konversi terhadap satuan ini, caranya sangat mudah.

Satuan panjang yang dipakai di seluruh dunia adalah meter.

Jika kita menaikkan satuan meter ke satuan di atasnya, maka nilainya harus dibagi 10.

Adapun jika kita menurunkan satuan meter ke satuan di bawahnya, makan nilainya harus dikalikan dengan 10.

Ilustrasinya adalah sebagai berikut:

1 m = 10 dm

1 m = 100 cm

1 m = 1000 mm

1 km = 10 hm

1 km = 100 dam

1 km = 1000 m

Untuk lebih jelasnya, kamu dapat melihat konversi satuan berdasarkan tangga konversi berikut:

Konversi satuan panjang

Konversi Satuan Berat

Sebenarnya, dalam fisika satuan berat yang benar adalah Newton. Akan tetapi dalam kehidupan sehari-hari, lebih sering berat diberi satuan kilogram.

Padahal seharusnya satuan kilogram tersebut diberikan untuk mengukur satuan massa. Massa tidak berubah di manapun kamu berada. Sementara besarnya berat akan berbeda-beda bergantung pada gaya gravitasi di tempat itu.

Tapi tidak apa-apa. Itu hanya masalah istilah saja. Selebihnya, dalam proses konversi nilai, keduanya sama saja.

Untuk mengkonversi satuan berat, sama seperti konversi satuan panjang di atas, kamu hanya perlu membagi dengan 10 setiap kenaikan satuan dan membagi dengan 10 setiap penurunan satuan.

33. Mengapa pada konversi dari Fahrenheit ke Reamur dikurangi 32


Karena pada saat dari Reamur ke farenheit itu pasti terakhirannya ditambah 32, jadi kalo mau dari Farenheit ke reamur harus di kurangi 32 dulu, contoh

32 R   = 9/4 x 32 + 32 = 72 + 32 = 104 F
104 F = 4/9 (104 - 32) = 4/9 x 72 = 32 R

36. Definisikan sebuah fungsi float konversi(suhu, asal, tuj), untuk mengkonversikan suhu dari Celsius ke Fahrenheit, Celsius ke Reamur, Fahrenheit ke Celsius, Fahrenheit ke Reamur, Reamur ke Celsius, dan Reamur ke Fahrenheit. Dimana suhu adalah suhu sumber, asal adalah satuan awal suhu yang akan dikonversi dan tuj adalah satuan hasil konversi


Dalam bahasa pemrograman Function atau fungsi adalah sebagian blok kode yang akan melakukan suatu tindakan tertentu. Cara pembuatan fungsi tersebut akan berbeda - beda pada tiap bahasa pemrograman, namun sebuah fungsi wajib mengembalikan sebuah nilai baik di bahasa pemrograman apa pun. Fungsi dapat menerima sejumlah parameter atau nilai masukan yang akan diolah dalam fungsi tersebut, namun hal ini hanya bersifat opsional yang mana artinya sebuah fungsi juga mungkin tidak memiliki parameter.

Dari soal tersebut dapat dilihat bahwa fungsi konversi yang akan dibuat mengembalikan nilai suhu yang telah dikonversi dengan tipe data float, dan memiliki 3 buah parameter yaitu suhu, satuan asal, dan satuan tujuan.

Maka function tersebut jika dibuat dalam java adalah

public static float konversi(float suhu, char asal, char tuj){

 float akhir;

 akhir = 0;  

 switch (asal) {

  case 'R':  

   if(tuj == 'F'){

    akhir = suhu * 2.25f + 32;

   }

   else if(tuj == 'C'){

    akhir = suhu / 0.8f;

   }

   break;

  case 'F':

   if(tuj == 'R'){

    akhir = (suhu-32)/0.44f;

   }

   else if(tuj == 'C'){

    akhir = (suhu-32) / 1.8f;

   }    

   break;

  case 'C':

   if(tuj == 'F'){

    akhir = suhu * 1.8f + 32;

   }

   else if(tuj == 'R'){

    akhir = suhu * 0.8f;

   }    

   break;  

 }  

 return akhir;

}


Pembahasan

Karena diminta sebuah fungsi float dengan 3 paramter masukan maka penulisan kode fungsi berikut akan seperti berikut

public static float konversi(float suhu, char asal, char tuj){

    .......................................

    .......................

}

Dan sebuah fungsi wajib mengembalikan sebuah nilai, sehingga baris kode

       return akhir;

adalah salah satu poin penting yang harus ada dalam sebuah fungsi.

Contoh pemanggilan fungsi tersebut kurang lebih seperti berikut

konversi(40,'R','C')

konversi(300,'F','C')

>Jawaban:

16° F = -8.89°C

>Penjelasan:

(16°F - 32) x (5/9) = -8.89°C

>Pembahasan:

Cara Konversi Dari Fahrenheit (°F) ke Celsius (°C)

Suhu S dalam derajat Celsius (°C) sama dengan suhu S dalam derajat Fahrenheit (°F) kurang 32, kali 5/9.  

S(°C) = (S(°F) - 32) × 5/9

39. Nyatakan 36°C dalam skala Fahrenheit, Reamur, Kelvin, dan Rankine


Jawaban:

Untuk mengkonversi suhu dari Celsius ke Fahrenheit, Reamur, Kelvin, dan Rankine, berikut adalah konversi yang tepat:

1. Fahrenheit (°F):

36°C = 96.8°F

2. Reamur (°Ré):

36°C = 28.8°Ré

3. Kelvin (K):

36°C = 309.15 K

4. Rankine (°R):

36°C = 554.67°R

Jadi, 36°C setara dengan 96.8°F, 28.8°Ré, 309.15 K, dan 554.67°R.
