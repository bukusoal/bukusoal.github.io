Jawab:
Jawaban d. 49

Penjelasan dengan langkah-langkah:

Diketahui baris aritmatika
{1, 3, 5, 7, ....}
Suku pertama (a)
{1, 3, 5, 7, ....}
--> a = 1
Perbedaan (b)
{1, 3, 5, 7, ....}
a, U₂
MAKA
b = U₂ - a
b = 3 - 1
--> b = 2
Suku ke-25
--> n = 25

Ditanya suku ke-25 (U₂₅)
Dengan rumus suku ke-n (Un)
Un = a + (n-1)b
Maka
U₂₅ = 1+(25-1)2
U₂₅ = 1+(24)2
U₂₅ = 1+48
U₂₅ = 49
Suku ke-25 adalah 49
