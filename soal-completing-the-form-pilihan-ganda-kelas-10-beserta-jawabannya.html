PEMBAHASAN1. Conditional Sentence type 1

➡️ Digunakan untuk menyatakan hasil dari kondisi yang mempunyai kemungkinan terjadi di masa yang akan datang karena kondisinya realistik untuk terjadi.

➡️Verbal sentence (yg ada kata kerja)

If + Simple present + Simple future If + (Subject + V1) + (Subject + will/ shall + V1)

➡️Nominal sentence (yg tak ada kata kerja)

If + Subject + be (is/am/are) + complement + Subject + will/ shall + be + complement

2. Conditional Sentence type 2

➡️ Digunakan untuk menyatakan hasil dari kondisi yang tidak mempunyai kemungkinan untuk terjadi karena jika dilihat kondisi di waktu sekarangnya tidak mungkin jadi agak sulit terjadi di masa depan.

If + Past Tense + Past Future TenseIf + (Subject + V2) + Subject + (would/could/might + V1)

3. Conditional Sentence type 3

➡️ Digunakan untuk menyatakan hasil dari kondisi yang jelas tidak ada kemungkinan untuk terjadi karena kondisinya harus sudah terjadi di masa lalu.

If + Past Perfect + Past Future PerfectIf + (Subject + had + V3) + (Subject + would + have + V3)

║PRESENT CONTINUOUS TENSE║

[tex]Answer[/tex]

Terapkan rumus present continuous

=> Subject+tobe(is/am/are)+verb-ing+ C  (positive)

=> Tobe(is/am/are)+Subject+verb-ing+ C  (interrogative)

Dialog 1

Sandra : Yes, I am. (1)

Rio        : (2) Is Roni coming ?

Sandra : No, He is not (3)

Rio       : Oh, I see. Are Santi and Mira coming? (4)

Dialog 2

Note :

Untuk menentukan -tobe (is/am/are), harap cermati bentuk subyek nya, apakah termasuk tunggal atau jamak.

[tex]Explanation[/tex]

PRESENT CONTINUOUS TENSE

Tenses ini digunakan untuk menjelaskan aktiftas/tindakan yang sedang dilakukan ketika dibicarakan. Pada umumnya, bentuk kata kerja-nya berakhiran -ing setelah tobe (is/am/are) untuk menunjukkan/menggambarkan aktifitas yang sedang berlangsung ketika dibicarakan.

Keterangan waktu ( adverb of time ) sebagai identifkasi tenses ini yaitu now, right now, still, at the moment.

Pattern/ Rumus

(+) Subject+tobe(is/am/are)+verb-ing+O/C

(-) Subject+tobe(is/am/are)+not+ verb-ing+O/C

(?) Tobe(is/am/are) + Subject + verb-ing+O/C?

Example

(+) My mother is watering the rose flowers now.

(-) My mother is not watering the rose flowers now.

(?) Is my mother watering the rose flowers now?

S + Have/has + V3 + O

Eat => Ate => Eaten

(+) S + Had + V3 + O/C(-) S + Had + not + V3 + O/C(?) Had + S + V3 + O/C?
