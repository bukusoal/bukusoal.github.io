Maka bukti induktif bahwa P(n) adalah benar untuk semua n ≥ q dilakukan melalui 2 (dua) langkah berikut:

a. Langkah awal: Tunjukkan bahwa P(q) adalah benar.

26. membuat soal dan pembahasan materi induksi magnet menjodohkan minimal 2 soal. ​


Berikut adalah 2 soal dan pembahasan materi induksi magnet:

Soal 1:

Sebuah kawat melingkar seperti lingkaran dilalui oleh arus DC. Jika diameter lingkaran ditingkatkan, apa yang akan terjadi pada gaya magnetik pada lingkaran tersebut?

Jawaban:

Jika diameter lingkaran ditingkatkan, gaya magnetik pada lingkaran akan berkurang. Ini karena gaya magnetik pada suatu kawat melingkar seperti lingkaran berkaitan dengan luas lingkaran. Semakin besar diameter lingkaran, semakin besar luas lingkaran dan semakin besar gaya magnetik yang diterima oleh lingkaran tersebut.

Pembahasan:

Gaya magnetik pada kawat melingkar seperti lingkaran dapat ditemukan dengan menggunakan hukum Faraday, yaitu V = -dΦ/dt, di mana V adalah voltase pada kawat, Φ adalah fluks magnetik dan t adalah waktu. Fluks magnetik pada suatu lingkaran dapat ditemukan dengan menggunakan rumus Φ = B * A, di mana B adalah intensitas magnetik dan A adalah luas lingkaran. Oleh karena itu, gaya magnetik pada suatu lingkaran berkaitan dengan luas lingkaran. Semakin besar diameter lingkaran, semakin besar luas lingkaran dan semakin besar gaya magnetik yang diterima oleh lingkaran tersebut.

Soal 2:

Sebuah bar magnet diposisikan dekat sebuah kawat melingkar. Apa yang akan terjadi pada arus pada kawat tersebut?

Jawaban:

Jika bar magnet diposisikan dekat kawat melingkar, arus pada kawat tersebut akan berubah. Ini karena adanya perubahan fluks magnetik pada kawat melingkar yang menyebabkan terjadinya gaya magnetik pada kawat tersebut dan menyebabkan arus pada kawat tersebut berubah.

Pembahasan:

Bar magnet diposisikan dekat kawat melingkar akan menyebabkan perubahan fluks magnetik pada kawat melingkar. Perubahan fluks magnetik akan menyebabkan terjadinya gaya magnetik pada kawat melingkar. Gaya magnetik pada kawat melingkar ini akan menyebabkan arus pada kawat tersebut berubah. Ini karena arus pada kawat melingkar dipengaruhi oleh gaya magnetik pada kawat tersebut. Oleh karena itu, jika bar magnet diposisikan dekat kawat melingkar, arus pada kawat tersebut akan berubah.


27. Buatlah 3 soal induksi matematika berserta jawabanya


Buatlah 3 soal induksi matematika berserta jawabannya

Buktikan bahwa: 1 + 2 + 3 + .... + n = ½ n (n + 1), untuk setiap n bilangan asli Buktikan pernyataan P(n) = n(n + 1)(n + 5) adalah bilangan kelipatan 3, untuk setiap n bilangan asli Buktikan bahwa 5ⁿ – 1 habis dibagi 4, untuk setiap n bilangan asli

Ada dua langkah dalam induksi matematika yaitu:

Buktikan bahwa untuk n = 1 benar Dengan mengasumsikan bahwa untuk n = k benar, maka buktikan bahwa untuk n = k + 1 juga benar

Pembahasan

1. Buktikan bahwa: 1 + 2 + 3 + .... + n = ½ n (n + 1), untuk setiap n bilangan asli

Pembuktian

Akan dibuktikan untuk n = 1 benar

n = ½ n (n + 1)

1 = ½ . 1 . (1 + 1)

1 = ½ . 1 . 2

1 = 1

(BENAR)

Jika untuk n = k benar yaitu

1 + 2 + 3 + .... + k = ½ k (k + 1)

Akan dibuktikan untuk n = (k + 1) juga benar

1 + 2 + 3 + … + k + (k + 1) = ½ (k + 1) ((k + 1) + 1)

|___________|

        ½ k (k + 1)   + (k + 1) = ½ (k + 1) (k + 2)

      ½ k (k + 1) + 1 (k + 1) = ½ (k + 1) (k + 2)

        (k + 1) (½ k + 1)         = ½ (k + 1) (k + 2)

        (k + 1) . ½ (k + 2)      = ½ (k + 1) (k + 2)

    ½ (k + 1) (k + 2)            = ½ (k + 1) (k + 2)

             |____________________|

                          Terbukti benar

Jadi terbukti bahwa 1 + 2 + 3 + .... + n = ½ n (n + 1), untuk setiap n bilangan asli

2. Buktikan pernyataan P(n) = n(n + 1)(n + 5) adalah bilangan kelipatan 3, untuk setiap n bilangan asli

Pembuktian

Akan dibuktikan untuk n = 1 benar

P(n) = n(n + 1)(n + 5)

P(1) = 1(1 + 1)(1 + 5)

P(1) = 1(2)(6)

P(1) = 12

(benar bahwa 12 kelipatan dari 3)

Jika untuk n = k benar yaitu

P(k) = k(k + 1)(k + 5) adalah bilangan kelipatan 3

Akan dibuktikan untuk n = (k + 1) juga benar

P(k + 1) = (k + 1) ((k + 1) + 1) ((k + 1) + 5)

P(k + 1) = (k + 1) (k + 2) (k + 6)

P(k + 1) = (k + 1) (k² + 6k + 2k + 12)

P(k + 1) = (k + 1) (k² + 8k + 12)

P(k + 1) = (k + 1) (k² + 5k + 3k + 12)

P(k + 1) = (k + 1) ((k² + 5k) + (3k + 12))

P(k + 1) = (k + 1)(k² + 5k) + (k + 1)(3k + 12)

P(k + 1) = (k + 1)k(k + 5) + (k + 1)3(k + 4)

P(k + 1) = k(k + 1)(k + 5) + 3(k + 1)(k + 4)

k(k + 1)(k + 5) adalah bilangan kelipatan 3 (berdasarkan n = k) 3(k + 1)(k + 4) sudah jelas merupakan bilangan kelipatan 3

Jadi

P(k + 1) = k(k + 1)(k + 5) + 3(k + 1)(k + 4) juga merupakan bilangan kelipatan 3

Jadi terbukti bahwa P(n) = n(n + 1)(n + 5) adalah bilangan kelipatan 3, untuk setiap n bilangan asli

3. Buktikan bahwa 5ⁿ – 1 habis dibagi 4, untuk setiap n bilangan asli

Pembuktian

Akan dibuktikan untuk n = 1 benar

5ⁿ – 1

= 5¹ – 1

= 5 – 1

= 4

(benar bahwa 4 habis dibagi 4ᵏ)

Jika untuk n = k benar yaitu

5ᵏ – 1 habis dibagi 4

Akan dibuktikan untuk n = (k + 1) juga benar

5ᵏ⁺¹ – 1

= 5ᵏ . 5¹ – 1

= 5ᵏ . 5 – 1

= 5ᵏ . (4 + 1) – 1

= 5ᵏ . 4 + 5ᵏ . 1 – 1

= 4 . 5ᵏ + 5ᵏ – 1

= (4 . 5ᵏ) + (5ᵏ – 1)

(4 . 5ᵏ) sudah jelas habis dibagi 4 (5ᵏ – 1) juga habis dibagi 4 (berdasarkan n = k)

Jadi (4 . 5ᵏ) + (5ᵏ – 1)  habis dibagi 4

Sehingga terbukti bahwa 5ⁿ – 1 habis dibagi 4, untuk setiap n bilangan asli
